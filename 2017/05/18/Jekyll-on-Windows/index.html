Jekyll on Windows

18 May 2017

1.安装Ruby
http://rubyinstaller.org/downloads/
2.安装DevKit
地址同上
在DevKit目录下：

ruby dk.rb init
ruby dk.rb install

3.安装Jekyll

gem install bundler
gem install jekyll

4.启动Jekyll

jekyll new myblog
cd myblog
jekyll serve

5.第一篇文章
在_post目录下以2017-5-18-BlogTitle.md的格式命名文章。
文章应使用“UTF-8无BOM格式编码”。
