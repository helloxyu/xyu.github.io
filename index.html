chrome-app-launcher-setup

2016-01-05
Apps
评论

不得不说，通过应用启动器直接打开Chrome应用真的很方便，可是不知怎么地，也不知道什么时候开始的，Google Chrome的启动器似乎从开始菜单中Chrome文件夹里消失了。这如何是好？
Step 1：建立快捷方式：
桌面上点击右键 – 新建 – 快捷方式 ：在弹出的对话框的“请输入对象的位置”处，输入以下信息：

“C:\Program Files (x86)\Google\Chrome\Application\chrome.exe” —show-app-list

(正常情况下，Chrome都是安装在C盘的)
点击下一步，在弹出的快捷方式名称输入框中，输入 “Chrome 应用启动器”。点击完成。


Step 2: 修改图标：
新建的快捷方式的图标跟Chrome的图标是一样一样的，为了加以区分，在图标上点击右键-属性-更改图标，选择一个看着舒服的图标就好啦。


Step 3: 放到任务栏
快捷方式上点击右键，选择“固定到任务栏”。
大功告成！

Nginx下配置WordPress的伪静态规则

2015-09-24
Web Server
评论

在Nginx目录下打开站点的配置文件（*.conf），在server{}里添加如下代码： 

location / {
if (-f $request_filename/index.html){
                rewrite (.*) $1/index.html break;
        }
if (-f $request_filename/index.php){
                rewrite (.*) $1/index.php;
        }
if (!-f $request_filename){
                rewrite (.*) /index.php;
        }
}

PS 若 


location /{}

已经存在，则直接添加括号内代码即可。

独立按键

2015-08-05
MCU
评论

按键在闭合和短开时，触点会存在抖动现象：

若要让独立按键正常工作，避免按下一次却出现了按下多次的效果，消除抖动是其中的一个非常关键的环节。以一次完整的按键过程（按下-提起-还原）为例，来描述消抖这一环节的实现： 

if (KEY != 0xff)                                    //当检测到有按键被按下时    
    {  
        Delay10ms();                                //通过延时函数度过按下独立按键是的抖动阶段  
        if (KEY != 0xff)                            //判断抖动阶段是否度过，按键是否处于被按下的稳定状态  
        {  
            Delay10ms();                            //再次延时，以判断按键是否还原  
            if (KEY == 0xff)                        //如果出现还原的状态，准备执行设定的效果  
            {  
                //Add the effect that you want    
                while (KEY != 0xff);                //当按键处于稳定的还原状态后，设定的效果开始执行  
            }  
        }  
    }
